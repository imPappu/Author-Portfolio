## ðŸ› ï¸ How to Use

### 1. Highlighted Text
In the **Customizer > Noble Hero Settings**, you can wrap any word in curly braces to style it.  
Example: `Redefining {Modern} Fiction` will make "Modern" italic and crimson.

### 2. Purchase Platforms
1. Go to **Purchase Platforms** in the WordPress menu.
2. Add a new platform (e.g., Amazon, Audible).
3. Set the title and upload a logo as the **Featured Image**.
4. When editing a Book, select this platform and enter the unique URL.

### 3. Hero Carousel
Upload up to 5 images in the Customizer to create a smooth, cinematic background transition.
